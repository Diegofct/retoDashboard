**Descripción:**

Este repositorio contiene un proyecto creado con Vite, React y TailwindCSS.

**Requisitos:**

-   Node.js
-   npm o yarn

**Instalación:**

1.  Clona el repositorio:

```
https://github.com/Diegofct/retoDashboard.git

```

2.  Instala las dependencias:

```
npm install

```

3.  Inicia el servidor de desarrollo:

```
npm run dev
